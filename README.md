## About

This is a real-time chat application built with React and Supabase, featuring AI integration with Google's Gemini.

## Features

- Real-time chat functionality
- User authentication
- Contact management
- Stories sharing
- AI-powered features
- Supabase backend integration

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the required environment variables in [.env](.env) with your Supabase credentials
3. Run the app:
   `npm run dev`
